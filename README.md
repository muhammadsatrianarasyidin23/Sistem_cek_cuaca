# Sistem_cek_cuaca
Program ini dibuat untuk membantu pengguna mengetahui kondisi lingkungan secara cepat dan praktis dan pengguna mendaptkan rekomendasi pakaian dan aktivitas yang cocok untuk suhu tersebut.
Dengan memasukkan angka suhu, system dapat mengklasifikasikan cuaca seperti dingin, sejuk, hangat, atau panas, serta memberikan informasi yang mudah dipahami.
